🎓 UBPOS 출결관리
Firebase Firestore와 연동해 실시간 출결을 관리합니다.
최초 1회 설정이 필요합니다.
STEP 1 — Firebase 프로젝트 생성
Firebase Console에서 프로젝트 이름 academy-attendance으로 생성 → Firestore Database 활성화 → 규칙을 테스트 모드(30일)로 설정
STEP 2 — 웹 앱 등록
프로젝트 설정(⚙) → 일반 → 내 앱 → 웹 앱(</>) 추가 → 앱 등록 → SDK 구성에서 firebaseConfig 객체 복사
STEP 3 — Config 붙여넣기
아래 필드에 복사한 JSON을 붙여넣고 저장하세요
